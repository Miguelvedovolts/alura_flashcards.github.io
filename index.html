Flash Cards de Estudo - Programação

O que é HTML?
Uma linguagem de marcação usada para estruturar o conteúdo da web.
O que é CSS?
Uma linguagem de estilo usada para descrever a aparência de um documento HTML.
O que é JavaScript?
Uma linguagem de programação que permite criar conteúdo interativo na web.
O que é uma função em JavaScript?
Um bloco de código projetado para executar uma tarefa específica.
O que é Git?
Um sistema de controle de versão distribuído para rastrear alterações no código.
O que é uma API?
Interface de programação de aplicativos que permite a comunicação entre sistemas.
